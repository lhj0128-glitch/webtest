# ⛺ 캠핑 준비물 체크리스트

날짜별로 캠핑 준비물을 체크하고 관리할 수 있는 웹 애플리케이션입니다.

## ✨ 주요 기능

- 📅 날짜별 체크리스트 관리
- ✅ 실시간 진행률 표시
- 💾 로컬 스토리지를 통한 데이터 저장
- 📱 반응형 디자인 (모바일 지원)
- 🗂️ 카테고리별 분류 (텐트, 타프, 침낭, 의자 등)

## 📋 체크리스트 항목

- 텐트 및 다크팩
- 타프 및 그라운드시트
- 침낭 및 매트
- 야계용 도구
- 버너 및 조리도구
- 의자 및 가구
- 물놀이 용품
- 무전기 및 전자기기
- 기타 캠핑 용품

총 **68개 항목**

## 🚀 사용 방법

1. **날짜 선택**: 캠핑 날짜를 선택합니다
2. **체크**: 각 항목을 체크하여 준비 상황을 표시합니다
3. **저장**: '저장' 버튼을 클릭하여 체크리스트를 저장합니다
4. **불러오기**: 저장된 날짜를 클릭하여 이전 체크리스트를 불러옵니다

## 💻 기술 스택

- HTML5
- CSS3 (Grid, Flexbox, Animations)
- Vanilla JavaScript (ES6+)
- Local Storage API

- 진행률 바

- 날짜별 관리 기능
