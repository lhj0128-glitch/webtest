# 💰 연말정산 계산기

간편하게 연말정산 예상 환급액을 계산할 수 있는 웹 애플리케이션입니다.

## ✨ 주요 기능

- 📊 2024년 기준 소득세율 적용
- 👨‍👩‍👧‍👦 인적공제 자동 계산
- 💳 신용카드/체크카드 소득공제
- 🏥 의료비, 교육비 세액공제
- 🏠 월세 세액공제
- 💼 연금저축 세액공제
- 📱 반응형 디자인

## 💡 주요 항목

### 인적공제
- 본인: 150만원
- 배우자: 150만원
- 부양가족: 1인당 150만원
- 자녀: 첫째 15만원, 둘째 15만원, 셋째부터 30만원

### 소득공제
- 국민연금, 건강보험: 전액 공제
- 신용카드: 15% 공제
- 체크카드: 30% 공제
- 현금영수증: 30% 공제
- 주택자금: 40% 공제

### 세액공제
- 보험료: 12% (한도 100만원)
- 의료비: 15% (총급여 3% 초과분)
- 교육비: 15%
- 기부금: 15%
- 월세: 12% (한도 750만원)
- 연금저축: 12% (한도 400만원)

## 📊 계산 프로세스

1. **총급여액** 입력
2. **근로소득공제** 자동 계산
3. **인적공제** 및 **소득공제** 적용
4. **과세표준** 산출
5. **소득세** 및 **지방소득세** 계산
6. **세액공제** 적용
7. **결정세액** 계산
8. **환급액/추가납부액** 확인

## ⚠️ 유의사항

- 이 계산기는 간이 계산용이며, 실제 연말정산 결과와 다를 수 있습니다
- 2024년 세율표 기준으로 계산됩니다
- 정확한 계산은 국세청 홈택스를 이용하세요
