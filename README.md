# 🎰 로또 번호 생성기

한국 로또(1-45 중 6개 번호) 자동 생성 웹 애플리케이션입니다.

## ✨ 주요 기능

- 🎲 1-45 범위에서 중복 없는 6개 번호 자동 생성
- 🎮 1-10개 게임 동시 생성 가능
- 📊 생성 통계 추적 (로컬 스토리지 사용)
- 🎨 반응형 디자인 (모바일 지원)
- ⚡ 애니메이션 효과

## 🚀 실행 방법

### 로컬 실행

1. 파일을 다운로드하거나 클론합니다
2. `index.html` 파일을 브라우저로 엽니다

또는 VS Code Live Server 사용:
```bash
# Live Server 확장 프로그램 설치 후
# index.html 파일에서 우클릭 > Open with Live Server
```

### GitHub Pages로 배포

1. GitHub 저장소 생성
2. 파일들을 푸시
3. Settings > Pages > Source를 `main` 브랜치로 설정
4. 생성된 URL로 접속

### Netlify로 배포

1. [Netlify](https://netlify.com)에 로그인
2. "New site from Git" 선택
3. 저장소 연결
4. 자동 배포 완료

### Vercel로 배포

1. [Vercel](https://vercel.com)에 로그인
2. "New Project" 선택
3. 저장소 import
4. 자동 배포 완료

## 💻 기술 스택

- HTML5
- CSS3 (Flexbox, Animations)
- Vanilla JavaScript (ES6+)
- Local Storage API

## 🎯 사용법

1. **게임 수 선택**: 1-10 사이의 숫자 입력
2. **번호 생성 클릭**: 로또 번호 자동 생성
3. **초기화**: 생성된 번호 지우기
